## 单选框

?> 单选框的 `name` 属性不能省略.

!> 单选复选的取值跟设置跟普通取值方式不太一样, 建议使用jquery 的 `prop` 去设置. `checkbox` 同样.

```js

// 设置
$(".radio").prop("checked",true);

// 取值
$(".radio").prop("checked");
```
