Defined in /Users/wilson/personal/htdocs/opensource/bui-rollup/src/scripts/core/bui.history.js:103

增加历史记录

Parameters:

  • option Object

    [对象]

    • id String

      [页面的唯一id, router.$ 会去取这个id作为父级的选择器 ]

    • pid String

      [模块名, 一般为路径名 ]

    • url String

      [页面跳转的模板地址 ]

    • replace Boolean

      [是否使用的替换历史记录 ]

    • param Object

      [从上一个页面传过来的参数 ]

    • part Object

      [当前页面里面有哪些局部加载的模块 ]

    • page Object

      [当前页面里面有哪些使用bui.page加载的模块 ]

    • toggle Object

      [当前页面使用的用来切换的对象, 必须有 hide show 方法,比方要在历史记录插入一个弹窗,那后退的时候,如果有这个对象,就会调用该对象的hide方法]

    • effect String

      [跟路由一致,默认为"push"]

    • syncHistory String

      [同步浏览器的路由, 默认 true ]

Returns:

Object:

option [返回对象]

Example:

方法:

  bui.history.add({
       id: "gid",
       pid: "pageid",
       url: "url",
       replace: false,
       param: {},
       part: {},
       page: {},
       toggle: null,
       effect: "push"
  });
